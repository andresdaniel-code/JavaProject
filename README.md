# JavaProject
Sistema CRUD con Java Web (Servlets, JDBC, MySQL)

Autor:
Nombre: Andres Daniel
Curso: ANALISIS Y DESARROLLO DE SOFTWARE
Fecha: 23/03/2026

Descripción:
Este proyecto es un sistema web desarrollado en Java que permite realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre usuarios.
Se implementa utilizando Servlets, JSP y conexión a base de datos mediante JDBC.

Tecnologías utilizadas:
- Java (JDK 26)
- Servlets
- JSP
- Apache Tomcat
- MySQL
- Maven

Funcionalidades:
- Registrar usuarios
- Listar usuarios
- Editar usuarios
- Eliminar usuarios

Base de datos:

Nombre de la base de datos:
usuarios_db

Tabla:
usuarios
(id, nombre, email, password, edad)

Acceso:
Ejecutar en navegador: http://localhost:8080/JavaProject/

Conclusión:
Este proyecto permite aplicar conceptos de desarrollo web en Java, conexión a bases de datos y arquitectura básica MVC mediante Servlets.
